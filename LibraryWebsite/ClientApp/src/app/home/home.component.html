Technology

Implemented

  • Code is hosted on Github
  • Backend is APS.NET Core 3.1 in C#
  • Frontend is Angular 8 with Typescript
  • User authentication and identity uses IdentityServer4
  • CI is done using CircleCI
  • Docker image of the application is built by CI
  • Unit tests are run for C# and Angular
  • Database tests are run for Entity Framework and it's migrations
  • End-to-end tests are run on CI in Docker, using Selenium in C#.
