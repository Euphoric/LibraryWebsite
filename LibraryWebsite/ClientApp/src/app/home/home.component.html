Welcome!

Welcome to library example application!

This is example application intended to showcase features of ASP.NET Core 3.0, Angular client, Docker and CI/CD behind it.

Features

Feature-wise the application is a simple library application.

Done

  • Displaying and editing books

Technology

Implemented

  • Code is hosted on Github
  • Backend is APS.NET Core 3.0 in C#
  • Frontend is Angular 7 with Typescript
  • CI is done using CircleCI
  • Docker image of the application is built by CI
  • Unit tests are run for C# and Angular
  • End-to-end tests are run on CI in Docker, using Selenium in C#.
